# 文本生成音频软件

* 由微软edge_ttk模块进行配音
* 利用tkinter模块实现窗口交互
* 由腾讯ide赋值编写

**有集成exe文件欢迎下载**

# **使用方法**


<img width="1366" height="705" alt="image" src="https://github.com/user-attachments/assets/35dafc92-761c-4765-8b2b-30cb418fbd35" />

- 选择好音色并输入好你要生成的文本
- 点击‘生成音频按钮’等待弹窗
- 当提示已生成后到桌面或所在文件夹去找output.mp3文件
